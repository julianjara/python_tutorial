## Ejemplo

```python
print ("hola que tal?")
x = 5
y = 2
print ("x + y = ", x + y)
print ("x - y = ", x - y)
print ("x * y = ", x * y)
print ("x / y = ", x / y)
print ("x // y = ", x // y)
print ("x % y = ", x % y)
print ("x ** y = ", x ** y)
```
Output

```bash
hola que tal?
x + y =  7
x - y =  3
x * y =  10
x / y =  2.5
x // y =  2
x % y =  1
x ** y =  25
```
